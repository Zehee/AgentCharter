## 设计方向（已确认）

| 维度 | 决策 | 说明 |
|------|------|------|
| **编号格式** | `{前缀}-NNN` | 如 `CORE-001`、`RULE-005`、`INST-002` |
| **分类** | 3 类 | 铁律（CORE）/ 规则（RULE）/ 实例（INST） |
| **实施方式** | HTML 注释 `<!-- RULE: ID -->` | 不改文档正文，不改结构，不改语气 |
| **框架气质** | ID 标签，不是条文重写 | 保持"操作手册"属性 |
